A modern Flutter-based calculator and multi-unit converter application that performs both mathematical calculations and unit conversions across multiple scientific and practical categories.

The application is designed with a clean mobile-first UI, allowing users to seamlessly switch between a calculator and a comprehensive converter system.

🏗 Built With

Flutter

Dart

Material Design

No backend is required — all conversions are handled locally within the Flutter application.


⚙️ Installation

1️⃣ Clone the repository
git clone https://github.com/MUGERWAIBRAHIM/flutter-calculator-converter.git

2️⃣ Navigate to project
cd calc

3️⃣ Install dependencies
flutter pub get

4️⃣ Run the application
flutter run

For web:

flutter run -d chrome
